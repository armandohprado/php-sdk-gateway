# GatewaySdk

## Installation

### Via Composer
Add the dependency to your ```composer.json```

```javascript
{
    "repositories": [
        //...
        {
            "type": "vcs",
            "url": "https://github.com/armandohprado/GatewaySdk"
        }
    ],
    //...
    "require": {
        //...
        "armandohprado/GatewaySdk": "0.*"
    }
}
```
